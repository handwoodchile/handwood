Place your logo image files in this folder.

Recommended file names:
- `logo.png` (primary, used by the site)
- `favicon.png` (optional small favicon)

Guidance:
- Use the exact PNG logo file you supplied; the site is configured to use `logo.png`.
- Recommended sizes: 1200×1200 (source), 512×512 (favicon), 200×200 (logo for header).
- Name the primary file `logo.png` so the site references `assets/logo.png` automatically.

If you provide the actual logo file, copy it here to replace the placeholder path used by the pages.
